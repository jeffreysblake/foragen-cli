# Foragen CLI Companion

The Foragen CLI Companion extension seamlessly integrates [Foragen CLI](https://github.com/jeffreysblake/foragen-cli). This extension is compatible with both VS Code and VS Code forks.

# Features

- Open Editor File Context: Foragen CLI gains awareness of the files you have open in your editor, providing it with a richer understanding of your project's structure and content.

- Selection Context: Foragen CLI can easily access your cursor's position and selected text within the editor, giving it valuable context directly from your current work.

- Native Diffing: Seamlessly view, modify, and accept code changes suggested by Foragen CLI directly within the editor.

- Launch Foragen CLI: Quickly start a new Foragen CLI session from the Command Palette (Cmd+Shift+P or Ctrl+Shift+P) by running the "Foragen CLI: Run" command.

# Requirements

To use this extension, you'll need:

- VS Code version 1.101.0 or newer
- Foragen CLI (installed separately) running within the VS Code integrated terminal
