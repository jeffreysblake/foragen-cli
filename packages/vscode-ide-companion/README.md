# Fora Code Companion

The Fora Code Companion extension seamlessly integrates [Fora Code](https://github.com/jeffreysblake/foragen-cli). This extension is compatible with both VS Code and VS Code forks.

# Features

- Open Editor File Context: Fora Code gains awareness of the files you have open in your editor, providing it with a richer understanding of your project's structure and content.

- Selection Context: Fora Code can easily access your cursor's position and selected text within the editor, giving it valuable context directly from your current work.

- Native Diffing: Seamlessly view, modify, and accept code changes suggested by Fora Code directly within the editor.

- Launch Fora Code: Quickly start a new Fora Code session from the Command Palette (Cmd+Shift+P or Ctrl+Shift+P) by running the "Fora Code: Run" command.

# Requirements

To use this extension, you'll need:

- VS Code version 1.101.0 or newer
- Fora Code (installed separately) running within the VS Code integrated terminal
